# Qr-Scan-Attendance-App



A modern and efficient QR Code-based Attendance Management System that allows users to mark attendance quickly by scanning QR codes. This system eliminates manual attendance and improves accuracy and speed.

🚀 Features
📷 QR Code-based attendance marking
👨‍🏫 Role-based system (Admin / Teacher / Student)
⚡ Real-time attendance tracking
📊 Attendance reports & analytics
🔐 Secure authentication system
📅 Date-wise and subject-wise attendance
📥 Export attendance (CSV / Excel) (optional)
📱 Responsive design (Mobile + Desktop)
